MarketSim

Qt 5.3 C++ QCustomPlot Qt widget

The aim of the simulator was to see how the density of the customers affects to the queue length and to the maximum waiting time. Additional goal was to create a rule which keeps the maximum waiting time in given tolerances by increasing and decreasing the amount of open cash registers. This simulator can be used as a tool to estimate the amount of service, space needed to meet the demand, amount of cash registers, space for the queues and the required amount of staff.

This project was an excellent opportunity to gain experience in GUI design, object-oriented programming in C++ and development in Qt.

IMDB Top 250 Checklist Web App

Java JSF 2.2 MySQL 5.6.14 Netbeans PrimeFaces 5.1 framework for JSF

KuvaOhjelma

C# WPF Visual Studio

The KuvaOhjelma is a photo selection application that concentrates on fast selection speed. The fast selection process is made possible by using keyboard shortcuts. The application was designed to use low system resources and to be an effective tool. The user can select a folder and the application will generate thumbnail view from all the photos inside the folder. Application was designed to be used with mouse and/or keyboard. Selected photos can be converted to another format or copied to different folder. Application has a full screen view with full functionality through shortcuts. As a keen photographer I designed this application to help filter photos more efficiently, even with just one hand in fullscreen mode.

Nokianpyry.fi is a website of Nokian Pyry sports club. The project started in 2004 and it finished in 2007. The website included a main site and separate child sites for every sport.
