MarketSim

Qt 5.3 C++ QCustomPlot Qt widget

The aim of the simulator was to see how the density of customers affects to queue length and the maximum waiting time, additionally to create a rule which keeps the maximum waiting time in given tolerances by increasing and decreasing the amount of opened cash registers. This simulator can be used as a tool to estimate the amount of service and space needed to meet the demand. Including amount of cash register, space for the queues and the amount of staff at given time.

The aim of this project was to gain experience in GUI design, object-oriented programming in C++ and development in Qt.

KuvaOhjelma

C# WPF Visual Studio

The KuvaOhjelma is a high speed photo selection application. Fast selection process is made possible by using keyboard shortcuts. The application was designed to use low system resources and to be an effective tool. User can select folder and application will load all photos from the folder to thumbnail view. Application was designed to be used with mouse and/or keyboard. Selected photos can be converted to another format or copied to different folder. Full screen view with full functionality through shortcuts. As a keen photographer I designed this application to help me filter my photos more efficiently, even with just one hand in fullscreen mode.
