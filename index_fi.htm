Figi.fm Android App

ADK Java RESTful API Scrum Eclipse Git

Figi.fm -palvelun Android-sovellus. Projektissa toimin osana viiden hengen ryhmää. Projekti kesti kahdeksan kuukautta. Sovellus toimii API-kutsujen avulla toimeksiantajan tuottaman RESTful-backendin kanssa. Projektin aikana kertyi kokemusta Java- ja Android-kehityksestä. Käytimme Asana- ja EasyBacklog-palveluita projektin sekä tehtävien hallintaan.

IMDB Top 250 Checklist Web App

Java JSF 2.2 MySQL 5.6.14 Netbeans PrimeFaces 5.1 framework for JSF

Web-sovellus, jolla käyttäjä voi seurata omaa edistymistään IMDB.com -sivuston 250 parhaan elokuvan listan läpi katsomisessa. Käyttäjälle esitetään lista 250 elokuvasta, jonne voidaan merkata katsotut elokuvat. Web-sovellus käyttää MySQL-tietokantaa tietojen tallentamiseen. Sovellus antaa käyttäjälle palautetta statistiikan ja diagrammien muodossa.

MarketSim

Qt 5.3 C++ QCustomPlot Qt widget

MarketSim on jonosimulaattori. Simulaattorissa voidaan säädellä monipuolisesti mm. asiakkaiden tiheyttä, kertoimia ja odotusaikaa. Simulaattori pitää jonon haluttua odotusaikaa lyhyempänä lisäämällä ja vähentämällä kassoja. Projektissa minua ovat motivoineet algoritmin luonti, Qt-kehityksen opiskelu ja käyttöliittymän suunnittelu.

KuvaOhjelma

C# WPF Visual Studio

KuvaOhjelma on kuvien katselu- ja valintaohjelma. Ohjelma avaa valitun kansion kaikki valokuvat esikatselunäkymään. Käyttäjä voi vaihtaa päänäkymän kuvaa hiiren tai näppäimistön avulla. Ohjelman avulla voidaan käydä nopeasti läpi isoja kuvajoukkoja ja valita halutut kuvat pikanäppäimen avulla. Kuvia voidaan tutkia koko näytön tilassa. Kuvat voidaan kääntää PNG-formaattiin. Ohjelma on suunniteltu omaan käyttöön, mahdollisimman kevyeksi ja nopeaksi apuvälineeksi valokuvien valitsemiseen.

Tietoa

Olen vuodenvaihteessa ohjelmistotekniikan insinööriopinnot loppuun saattanut ohjelmoija. Web-sivustojen ja C++-kielen kanssa tehnyt harrastelija projekteja vuodesta 2001. Kehitin ja ylläpidin vuosina 2004-2006 osuma.org ja nokianpyry.fi -sivustoja (PHP, MySQL).

Hiljattain työskentelin Figi.fm -palvelun Android-sovellus projektissa ja olin suunnittelemassa filmiarkiston hallintajärjestelmää Thaimaan kansalliselle filmiarkistolle. Tällä hetkellä harralistelija projektina on VR kokeiluja Unity3D-pelimoottorilla.

Etsin aina uusia tekniikoita ja teen harrastuspohjalta pieniä projekteja. Alan ulkopuolella viihdyn valokuvauksen parissa ja reissun päällä.
